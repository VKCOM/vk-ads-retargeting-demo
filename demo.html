Ретаргетинг ВКонтакте

Эта страница на примерах поможет разобраться, как пользоваться JS API Ретаргетинга ВКонтакте.

Описание JS API Простой пример

В этом примере используются все методы JavaScript API: простое событие посещения страницы, событие нажатия на кнопку, и добавление в аудиторию по её ID. Исходный код с комментариями доступен на github в файлах demo.html и demo.js.
