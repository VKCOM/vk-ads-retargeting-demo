Ретаргетинг ВКонтакте

Эта страница на примерах поможет разобраться, как пользоваться JS API Ретаргетинга ВКонтакте.

Описание JS API Простой пример

В этом примере имитируется распространённый сценарий: заполнение пользователем формы на сайте. Используются все методы JavaScript API: простое событие посещения страницы, событие нажатия на кнопку, и добавление в аудиторию по её ID. Исходный код с комментариями доступен на github в файлах demo.html и js/demo.js.

При загрузке страницы вызывается метод Hit. При изменении любого из полей формы — Add. При нажатии на кнопку — Event.
